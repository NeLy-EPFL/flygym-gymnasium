Simulation
==========

The user interacts with the simulated fly via the ``Simulation`` class, which implements the `Gym <https://gymnasium.farama.org/>`_ ``Env`` API. Additionally, we have provided a ``SingleFlySimulation`` class — a shorthand wrapper for simulations with only one fly.
